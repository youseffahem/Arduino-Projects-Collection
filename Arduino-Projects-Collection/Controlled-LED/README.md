# Controlled LED

A simple Arduino project that controls an LED using a push button.

## Components
- Arduino Uno
- Breadboard
- LED
- Push Button
- 220Ω Resistor
- Jumper Wires

## Features
- Turn LED ON when button is pressed
- Turn LED OFF when button is released
- Basic digital input/output control

## Circuit
The push button is connected to pin 2 and the LED is connected to pin 13.

## How It Works
The Arduino continuously reads the state of the push button.
When the button is pressed, the LED turns on.
Otherwise, the LED remains off.
