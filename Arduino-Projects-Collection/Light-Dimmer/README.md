# Light Dimmer

Arduino PWM project for controlling LED brightness using a potentiometer.

## Components
- Arduino Uno
- LED
- Potentiometer
- 220Ω Resistor
- Breadboard
- Jumper Wires

## Features
- Adjustable LED brightness
- PWM signal control
- Real-time analog input reading

## Circuit
The potentiometer is connected to analog pin A0.
The LED is connected to PWM pin 9.

## How It Works
The Arduino reads the potentiometer value from A0.
The value is converted into PWM output to adjust LED brightness.

## Technologies Used
- Arduino IDE
- Embedded C/C++
- PWM Control
